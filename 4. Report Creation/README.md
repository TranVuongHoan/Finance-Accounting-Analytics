# 4. Report Creation

## Interactive Dashboard

The file `finance_dashboard.html` is a fully self-contained interactive dashboard. Open in any web browser.

## Dashboard Sections

### Section 1 — Data Quality
- Missing payment dates, PO numbers, contracts
- Unusual VAT rate count
- Data completeness gauges

### Section 2 — Overdue Crisis
- Overdue rate by department (bar)
- Overdue rate by transaction type
- Monthly overdue trend
- Top 15 most overdue transactions (table)

### Section 3 — Budget Variance
- Variance by quarter (trend line)
- Variance by department (bar)
- Distribution of variance % (histogram)

### Section 4 — Tax & Compliance Risk
- Tax Payment overdue + rejection rates
- Approval status breakdown (doughnut)
- Payment method efficiency
